Program Transformasi Linier ini akan menerima input yang menjalankan sub-program untuk 2 dimensi atau 3 dimensi.
Program ini akan menampilkan bentuk yang dibuat dan juga dapat menjalankan fungsi-fungsi dari transformasi linier
Fungsi-fungsi yang dapat digunakan pada transformasi linier ini adalah:
1. Translasi
2. Dilatasi
3. Rotasi
4. Refleksi
5. Shear (Pergeseran)
6. Stretch (Peregangan)
7. Custom Transformasi
8. Reset
Selain itu, dapat juga dilakukan input transformasi beberapa buah dengan meng-input command multiple.
Terminasi dari program ini dapat dilakukan dengan meng-input exit

Sub-Program dari 2 dimensi ini akan menerima jumlah titik penyusun bidang tersebut. Setelah itu akan juga diinput nilai dari titk-titik tersebut dimana peng-inputan dari titik tersebut harus berurutan sehingga dapat dibentuk bentuk yang diharapkan
Pada sub-program 2 dimensi ini, dapat dilakukan input untuk melaksanakan dari fungsi-fungsi transformasi linier yang tersedia.


Sub-Program dari 3 dimensi ini berupa bentuk dari Kubus. Pada Sub-Program ini, dapat dilakukan pergeseran kamera sehingga dapat melihat dari sisi depan, belakang, atas, bawah, kiri dan kanan. Tombol-tombol yang dapat menggerakan kamera ini dengan menginput "w","a","s","d","z","c".
Tombol lain yang dapat digunakan adalah "f" dimana setelah menekan tombol ini, maka dapat dilakukan input fungsi dari transformasi.
Fungsi dari Transformasi Linier pada sub-program ini juga sama dengan fungsi transformasi linier pada 2 dimensi.

Untuk menjalankan program : buka folder bin -> dist -> main -> main.exe
