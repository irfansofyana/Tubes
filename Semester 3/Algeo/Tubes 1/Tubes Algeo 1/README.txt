README

	SPL adalah program yang menyelesaikan persoalan mengenai Sistem Persamaan Linear (SPL) dan Interpolasi Polinom. 
	Program ini mengimpelementasikan eliminasi Gauss, dan eliminasi Gauss-Jordan untuk memecahkan persoalan.
	Panduan dari program ini bisa dilihat di bawah ini. Pembuatan dari program ini dibuat menggunakan bahasa pemograman
	Java dan sudah melalui serangkaian tes untuk mengujinya.

	Requirements :
	- Java SDK

	Cara menggunakan program ini :
	1. Buka CMD.exe di dalam folder bin
	2. Ketikkan java SPL
	3. Pilih menu yang anda inginkan, jika anda ingin mencari solusi dari suatu SPL pilih 1. Jika ingin mengetahui interpolasi polinom maka pilih 2. atau pilih 3 untuk keluar dari program
	4. Pilih metode yang anda inginkan. Gauss atau Gauss-Jordan
	5. Jika anda mempunyai file yang ingin anda input maka, taruh file tersebuh di bin. Lalu, ketikkan namafile yang kamu ingin input.
	6. Jika anda ingin menginput dari keyboard, cukup tulis "-" ke program.
	7. Program akan menampilkan solusi dari input yang sudah anda masukkan.
	8. Selesai

	Troubleshooting :
	1. Input MissMatch
		Anda salah memasukkan file. File yang anda masukkan tidak sesuai format. Solusi : Baca Dokumentasi untuk mengetahui file eksternal yang benar
	2. File not Found
		File anda tidak dapat dibaca. Solusi : Perhatikan penulisan nama file ke program, format dari file tersebut  .txt atau bukan.
